# GameBuy
GameBuy is a website that uses CheapShark API to find best deals on PC video games. GameBuy also makes use of Bing Image Search API to display related images.

## Tools and Technologies Used
   * Front-End
     - HTML
     - CSS
     - JavaScript
   * Back-End
     - Node.js
        - Express
     - CheapShark API
